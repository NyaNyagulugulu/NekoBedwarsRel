# 梦幻次元 BedwarsRel 插件
不改名和版本号是为了兼容哪些基于BwRel制作的附属插件

## 项目简介
梦幻次元 BedwarsRel 是一款基于 Minecraft 的起床战争游戏插件，为玩家提供刺激有趣的床战体验。玩家需要保护自己的床不被破坏，同时摧毁其他队伍的床来获得胜利。

## 功能特色
- **多队伍对战**：支持多支队伍同时对战
- **资源生成**：地图中有资源生成器，玩家可收集资源购买装备
- **商店系统**：村民商店提供各种装备和道具
  ## 快捷购买系统
### 标准快捷购买区域
- 位于商店界面第4行
- 共9个槽位，所有玩家均可使用
- 可通过Shift+点击商店物品来设置
- 可通过Shift+点击槽位来移除设置

### 扩展快捷购买区域
- 位于商店界面第5行
- 共9个额外槽位（槽位10-18）
- 仅限VIP和MVP及以上权限玩家使用
- 无权限玩家将看到红色提示界面
- 有权限玩家可正常使用所有功能
- **团队合作**：需要团队协作来保护床和攻击敌人
- **统计数据**：记录玩家的游戏统计数据
- **二次元风格**：独特的二次元界面设计，包括梦幻次元主题的Tab列表

## 安装说明
1. 将 `BedwarsRel-1.3.6.jar` 文件放入服务器的 `plugins` 文件夹
2. 重启服务器以生成配置文件
3. 根据需要修改 `config.yml` 中的配置
4. 重启服务器使配置生效

## 配置说明
- **数据库配置**：支持 MySQL 数据库存储统计数据
- **Tab列表样式**：已设置为梦幻次元二次元风格
- **游戏模式**：可配置不同队伍数量和地图设置
- **资源生成**：可自定义资源生成间隔和类型

## 指令列表
- `/bw` - 基础指令
- `/bw join [游戏名] [队伍]` - 加入游戏
- `/bw addgame [游戏名]` - 添加游戏
- `/bw setregion [游戏名] [区域]` - 设置游戏区域
- `/bw setspawn [游戏名] [队伍]` - 设置队伍出生点
- `/bw setbed [游戏名] [队伍]` - 设置队伍床的位置
- `/bw save [游戏名]` - 保存游戏设置
- `/bw start [游戏名]` - 开始游戏
- `/bw stats` - 查看统计数据

## 权限列表
- `bw.vip` - 解锁扩展快捷购买区域等VIP功能
- `bw.mvp` - 解锁扩展快捷购买区域等MVP功能
- `bw.setup` - 管理员设置权限
- `bw.cmd` - 使用插件指令权限

- 可选：HolographicDisplays 用于全息显示
- 可选：Multiverse-Core 用于多世界支持

## 二次元风格特性
- Tab列表显示：采用梦幻次元主题的二次元风格
- 颜色搭配：使用粉色、蓝色等二次元常用配色
- 装饰符号：添加了心形、箭头等二次元装饰符号

## 数据库设置
插件支持 MySQL 数据库存储玩家统计数据：
- 配置数据库连接参数
- 自动创建数据表
- 持久化存储玩家游戏数据

## 常见问题
1. **插件无法加载**：检查服务器版本是否为 1.12.2
2. **数据库连接失败**：检查数据库配置是否正确
3. **游戏无法开始**：确保设置了必需的游戏参数
4. **扩展快捷购买区域无法使用**：扩展快捷购买区域（第5行）仅限VIP/MVP及以上权限玩家使用
5. **快捷购买显示"物品不存在"**：可能是物品配置错误，请联系管理员
6. **无法设置快捷购买**：检查数据库连接是否正常，或联系管理员
